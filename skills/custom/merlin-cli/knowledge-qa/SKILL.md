# 知识问答（通用）

## 摘要

当用户提出使用类问题（如如何配置队列、如何查看日志、常见报错解释、最佳实践）或其他技能不适用时，使用本技能。自动检索知识库并综合已有文档与案例，输出明确答案、依据与引用链接。

## 适用场景

- 用户提出“如何…/为什么…/能否…”等使用或概念性问题。
- 问题无法映射到具体任务 ID/设备 ID，或不涉及代码/资源的即时改动。
- 其他技能（如任务排查/Devbox 故障）不适用或无法直接执行。

## 前置条件

- `merlin-cli knowledge search` 可用，用以按关键词检索知识库。
- 可选：Bash 工具可用于打开参考链接或做轻量验证（如 `curl`）。

## MCP 工具使用

当 MCP 工具不可用时，可以使用 merlin-cli CLI 作为替代。merlin-cli 会动态从 MCP 服务获取所有可用工具。

```bash
# 检查 merlin-cli 是否已安装，如未安装则下载
# 检查 merlin-cli 是否已安装
merlin-cli --help &>/dev/null

# 如未安装，执行以下命令下载安装
curl -fsSL https://ml.bytedance.net/api/agent/system/tos-proxy/merlin-cli/latest/install.sh | bash

# 查看所有可用工具
merlin-cli list-tools

# 按关键词过滤工具
merlin-cli list-tools --filter <keyword>

# 查看特定工具的帮助信息
merlin-cli <tool-name> --help

# 调用工具示例
merlin-cli knowledge search --json '{"query": "如何配置队列", "top_k": 10}'
```

如果出现认证错误（401/403），请运行：`merlin-cli login`

## 输入

- 自然语言问题文本；可附带上下文（平台、模块、错误关键词、期望目标）。

## 输出

- 结构化回答：直接答案、关键点、依据说明、引用列表、下一步建议。

## 步骤

1.  **识别问题类型**
    - 分类：使用指南、配置说明、常见错误解释、最佳实践、对比/选择建议、术语定义。
    - 判断是否存在更合适技能：若问题明显属于任务/Devbox 故障，请引导至对应技能；否则进入检索。

2.  **构造检索关键词并查询**
    - 提取问题中的关键术语、模块名、错误名、参数名；加入同义词与中英词形（如 `epoch/轮次`）。
    - 调用 `merlin-cli knowledge search`，得到候选文档与摘要。

3.  **筛选与综合**
    - 依据新近性、来源权威度、与问题匹配度进行排序；去重相似内容。
    - 对相互矛盾的说法进行比对，优先采用官方/平台文档或高质量案例。

4.  **生成结构化答案**
    - 直接回答：用清晰简洁语言给出结论或操作步骤。
    - 关键点：列出核心注意事项/前提条件/常见坑。
    - 依据说明：简单解释判断逻辑或引用内容的理由。
    - 引用列表：提供 2–5 个高相关链接或文档标识，带标题与来源。
    - 下一步建议：如需进一步操作，指出应使用的技能或具体动作。

5.  **可选轻量验证**
    - 对涉及命令或接口的答案，可做最小化验证（不修改用户资源）。例如示例请求、参数校验。

## 结构化输出模板

- 回答：<一句话或分步说明>
- 关键点：
  - <要点 1>
  - <要点 2>
- 依据：<简要判断依据>
- 引用：
  - [标题 1](<链接或文档ID>) 来源：<站点/模块>
  - [标题 2](<链接或文档ID>) 来源：<站点/模块>
- 下一步：<建议的后续技能/动作>

## 触发规则（强约束）

- 满足任一条件即触发本技能：
  - 用户问题不包含可操作的 `job_id/devbox_id` 且不需要修改资源/代码。
  - 问题为“如何使用/常见报错解释/概念与参数说明/策略对比”。
  - 其他技能评估后不适用或无法收敛到具体自动化操作。

## 约束与注意事项

- 优先给出可执行的、简明的答案；必要时提供分步指引。
- 避免臆测：对不确定点要明确标注“不确定”，并给出可验证路径或参考文档。
- 引用内容需保留来源与标题；如含敏感信息，应适度脱敏或摘要化。
- 多语言兼容：问题与答案尽量与用户语言保持一致；必要时提供术语的中英对照。

## 失败与兜底

- 若检索结果不足或相互矛盾：输出当前最佳判断与不确定点，列出需要补充的信息项，避免误导。
- 若工具不可用或网络异常：告知限制与临时建议（如稍后重试、查看本地文档）。
